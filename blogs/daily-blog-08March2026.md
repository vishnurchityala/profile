# Revamping Some Old Projects : Profile Website

After a long of time of procrastinating i finally decided to revamp my portfolio website, my requirements were simple a sweet looking HTML, CSS website which just delivers information and gives a platform to rant daily about my programming updates. So here i am !!

## Core Idea for Website

Pinterest got me the main inspiration for this website, i so a Adobe designers custom resume poster where he designed his website in minecraft theme and tailored his portfolio webpage around that theme. I wanted similar design style, from past few weeks i was testing out Google-Gemini-Flash for various image generation task where i have created alot of usable assets thinking of sharing them publically later !!!, so getting back to the point i started create design pallette and STYLE.md file using GPT then after i decided this color pallette mixture of dark and light colors created whole STYLE.md guide file.

## Vibe-Coding for Website (GPT-Codex)

Don't ask how but i have access to un-limited GPT codex platform so i have GPT this prompt:

```text

Hey Codex i am planning to re-vamp my personnel portfolio project, keeping the structure exactly same HTML, CSS, JS project no overhead of framework for single web-page.

design guidelines: minecraft inspired UI style with brown, orange, beige as primary colors. 
robotic fonts with grid style background images, plain square shaped edges with sharp corners and black shadows on hover. 
use font-awesome and direct cdn links for icons and typography don't download any asset

dino theme: i have few dino's in my dinos/ folder use them to tailor my website with a playable dino parade.

source of information: checkout my two resumes in resume/ folder and take the content for project

blogs page: i have one blogs folder where i will document my blogs in md file so write js such that automatically that md file is rendered in HTML.

keep the code simple and documented.

```

## Iterations of Codex

Obviously Codex is not so great like anti-gravity for frontend, it took my approx 10-15 prompts to get this current website, lot of overlaying issues and required manual fixing, physics got missing in dino's parade, but indeed this is the best work anyone can get form AI workflow, documenting designs and CSS guidelines in MD file is very cool trick.
